【Android】Lab5_Appwidget及Broadcast使用

发表于 2018-05-12 | 分类于 Android | | 阅读次数: | 字数统计: 976

¶ 一、实验题目

appwidget 及broadcast 使用

¶ 二、实验目的

  1. 掌握 AppWidget 编程基础
  2. 掌握 Broadcast 编程基础
  3. 掌握动态注册 Broadcast 和静态注册 Broadcast

¶ 三、实现内容

实现一个Android 应用，实现静态广播、动态广播两种改变widget 内容的方法。在上次实验的基础上进行修改，所以一些关于静态动态广播的内容会简略。

具体要求:

  1. 点击widget可以启动应用，并在widget随机推荐一个商品
  2. 点击widget跳转到该商品详情界面
  3. 点击购物车图标，widget相应更新
  4. 点击widget跳转到购物车界面
  5. 实现方式要求:启动时的widget的更新通过静态广播实现，点击购物车图标时候widget的更新通过动态广播实现。
